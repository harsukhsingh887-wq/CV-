Personal Resume Website

This project is a personal resume/CV website built using HTML and CSS during the initial phase of my Web Development elective. It represents my first step into building structured and responsive web pages.

About the Project

The goal of this project was to:

* Understand the fundamentals of HTML structure
* Learn how to style webpages using CSS
* Create a clean and visually appealing resume layout
* Practice building a multi-section webpage with navigation

This website includes all essential sections of a professional resume and is fully responsive for different screen sizes.

 Technologies Used

* HTML5
* CSS3
* Basic Responsive Design (Media Queries)

Features

* Smooth navigation bar with section links
* Responsive layout for mobile and desktop
* Profile section with image
* Structured resume sections:

  * About Me
  * Education
  * Skills
  * Projects
  * Experience
  * Personal Details
  * Contact Information
  * Clean UI with modern styling and hover effects


What I Learned

* Structuring content using semantic HTML
* Styling layouts using CSS (Flexbox, spacing, shadows)
* Making a website responsive
* Improving UI/UX with small design enhancements
